
Implementazione di un server CRUD per MotorTG.it

| Characteristic |                                                                                           |
|----------------|-------------------------------------------------------------------------------------------|
| Language       | TypeScript                                                                          |
| Database       | Postgres, with the [Postgres adapter](https://socket.io/docs/v4/postgres-adapter/)        |
| Cluster?       | Not needed |

## Usage

```
$ docker-compose up -d
$ npm install
$ npm start
```
